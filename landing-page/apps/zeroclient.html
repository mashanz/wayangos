WayangZeroClient

Turn any PC into a thin client. Boot from ISO, connect to your remote desktop via RDP or VNC. No OS installation required — just boot and work.

Features

📺

Framebuffer UI

Runs directly on /dev/fb0. No X11, no Wayland, no desktop environment needed for the launcher. Minimal resource usage.

🔗

RDP & VNC Support

Connect to Windows, Linux, or macOS remote desktops. Uses FreeRDP for RDP and TigerVNC for VNC connections.

📋

Connection Manager

Save, edit, and organize multiple remote connections. Full keyboard navigation with intuitive controls.

🔄

Auto-Reconnect

Automatic reconnection with configurable retries. Keeps sessions alive through network interruptions.

🔒

Kiosk Mode

Boots straight into the client interface. No shell access, no escape — perfect for public terminals and shared workstations.

⚡

Tiny Footprint

Single static binary under 200KB. The entire bootable ISO is under 65MB including kernel and all dependencies.

Build from Source

WayangZeroClient is a single C file. Compile it with any GCC that supports static linking.

gcc -static -O2 -Wall -o wayang-zeroclient zeroclient.c -lm -lpthread

Configuration

Connections are stored in /data/connections.conf:

[My Server] name=My Server type=rdp address=192.168.1.100 port=3389 username=admin password=secret resolution=auto
