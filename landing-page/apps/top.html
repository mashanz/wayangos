Live system dashboard — CPU, memory, network, and disk at a glance

Your System, Visualized

Real-time performance data, beautifully displayed.

🏎️

Speedometer Gauges

Analog-style gauges for CPU and memory usage. Instantly see how hard your system is working — no numbers to decipher.

🐝

CPU Honeycomb

Each CPU core displayed as a hexagonal cell, color-coded by load. See at a glance which cores are busy and which are idle.

📈

Live Network Graphs

Scrolling line graphs showing upload and download speeds in real-time. Watch your network activity as it happens.

💾

Disk Activity

Live read/write graphs for your storage devices. Spot bottlenecks and see exactly when your disk is being hit hard.

📋

Process Manager

Sortable list of running processes with CPU, memory, and PID. Find resource-hungry processes and stop them if needed.

⚡

Always Up-to-Date

All metrics refresh continuously. Lightweight enough to run alongside your other apps without slowing things down.
