WayangExplorer

Dual-pane file explorer with directory listing and file preview. Keyboard-driven navigation, file operations, and text preview — all on the framebuffer.

Features

📂

Dual-Pane Layout

Left pane shows directory listing, right pane shows file details and preview. Navigate efficiently between directories.

⌨️

Keyboard-Driven

Full keyboard navigation. Arrow keys to browse, Enter to open, Backspace to go up. No mouse needed.

📋

File Operations

Copy, move, delete, and rename files and directories. All operations with confirmation dialogs to prevent accidents.

📄

Text File Preview

Preview text files directly in the right pane. Quick way to inspect configs, logs, and scripts without opening an editor.

📺

Framebuffer Rendering

Renders directly to /dev/fb0. No X11, no Wayland, no desktop environment. Works on any Linux system with a framebuffer.

⚡

Static Binary

Single static binary with zero runtime dependencies. Drop it on any system and run. Under 200KB compiled size.

Keyboard Shortcuts

KeyAction
↑ / ↓Navigate up/down in file list
EnterOpen directory / preview file
BackspaceGo to parent directory
TabSwitch between panes
cCopy selected file/directory
mMove selected file/directory
dDelete selected file/directory
rRename selected file/directory
qQuit / exit explorer

Build from Source

WayangExplorer is a single C file. Compile it with any GCC that supports static linking.

git clone https://github.com/mashanz/wayangos-explorer.git cd wayangos-explorer gcc -static -O2 -Wall -o wayang-explorer explorer.c -lm -lpthread

Or download the pre-built ISO from the Downloads page.
